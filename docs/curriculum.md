# Curriculum Map

FDE API Academy is organized around practical engineering skills, not competitive programming.

1. Python Refresher
2. Data Parsing Fundamentals
3. JSON Mastery
4. HTTP Fundamentals
5. GET Requests
6. POST Requests
7. PUT and PATCH Requests
8. DELETE Requests
9. Authentication
10. GitHub API Deep Dive
11. Data Transformation
12. CSV and Data Export
13. SQLite Basics
14. Debugging Integrations
15. Real API Integrations
16. Automation Workflows
17. FDE Interview Simulations

Each lesson includes:

- Explanation
- Visual example
- Example code
- Guided exercise
- Challenge exercise
- Automatic validation
- Reflection questions
- Mentor hint ladder
